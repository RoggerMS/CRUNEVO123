# Roadmap CRUNEVO

## Fase 1: MVP (Actual)
- Infraestructura base (Monorepo, Docker).
- Autenticación y Usuarios.
- Documentos (CRUD + Uploads).
- Q&A (Preguntas y Respuestas).
- Feed Básico.
- Admin Básico (Reportes).
- Stubs para IA y Cursos.

## Fase 2: Social & Gamification (Futuro)
- Comentarios en documentos.
- Sistema de puntos/reputación.
- Perfiles públicos mejorados.

## Fase 3: Contenido Premium (Futuro)
- Módulo de Cursos real (Streaming).
- Marketplace de contenidos.
- Integración real de IA.
